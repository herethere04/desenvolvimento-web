Exemplo 4 HTML - Elementos HTML (Listas ordenadas e não ordenadas)

Listas

Listas em HTML são representadas pelas tags <ol>(lista ordenada) e <ul>(lista não ordenada). Essas listas possuem diversas finalidades, podendeo ser ultilizadas para exibir informações ou simplismente serem usadas em menu

Toda vez que usarmos listas ordenadas ou não ordenadas deveremos ultilizar a tag <li>

Exemplo de lista ordenada <ol>

  1. item de lista 1
  2. item de lista 2
    1. Subitem 2.1
    2. Subitem 2.2
    3. Subitem 2.3
    4. Subitem 2.4
  3. item de lista 3
  4. item de lista 4
  5. item de lista 5
    1. Subitem 5.1
    2. Subitem 5.2
    3. Subitem 5.3
      1. Subitem 5.1.1
      2. Subitem 5.2.2
      3. Subitem 5.3.3
      4. Subitem 5.4.4
    4. Subitem 5.4
  6. item de lista 6

Exemplo de lista não ordenada <ul>

  • item de lista 1
  • item de lista 2
    • Subitem 2.1
    • Subitem 2.2
    • Subitem 2.3
    • Subitem 2.4
  • item de lista 3
  • item de lista 4
  • item de lista 5
    • Subitem 5.1
    • Subitem 5.2
    • Subitem 5.3
      • Subitem 5.1.1
      • Subitem 5.2.2
      • Subitem 5.3.3
      • Subitem 5.4.4
    • Subitem 5.4
  • item de lista 6
