Exemplo 11 - CSS - Formas de Vinculação

Vinculação Interna (Inline)

Nesse método, o CSS é escrito diretamente em uma tag html usando o atributo style. Para isso, devemos adicionar o atributo style ao elemento html que desejamos estilizar.

Nesta página estamos usando a vinculação inline.

Este tipo de vinculação deve ser evitado, pois dificulta a manuntnção da página e so deve ser usado quando necessitamos estilizar unicamente um elemento.
