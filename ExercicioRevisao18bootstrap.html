Exemplo 18 - Revisão sobre BootStrap

O que é o BootStrap

O Bootstrap é o framework front-end mais popular do mundo, utilizado para criar sites e aplicações web responsivos de forma rápida e fácil. Ele fornece uma coleção de componentes CSS, HTML e JavaScript pré-estilizados (como botões, formulários e navegação) que se adaptam automaticamente a diferentes tamanhos de tela (mobile-first).

BootStrap Cores Contextuais

As cores contextuais no Bootstrap 5 são classes utilitárias baseadas em temas que aplicam cores a textos, fundos, bordas e componentes, indicando uma intenção específica (como sucesso, perigo, alerta). Elas facilitam a criação de interfaces consistentes e a manutenção do código.

BootStrap Contêiner e Contêiner-fluid

No Bootstrap, containers são os elementos fundamentais para envolver e alinhar o conteúdo. A principal diferença entre eles é como se comportam em relação à largura da tela.

BootStrap Contêiner

1 . container (Largura Fixa Responsiva) A classe .container define uma largura máxima (max-width) fixa para cada tamanho de tela (breakpoint).

  • Comportamento: Ele "pula" de tamanho conforme a tela aumenta ou diminui.
  • Visual: Em telas grandes, o conteúdo fica centralizado com margens nas laterais.
  • Uso ideal: Layouts "box" (encaixotados) onde você quer que o conteúdo tenha limites claros e centralizados.

Uso do Container-Fluid

A classe .container-fluid no Bootstrap cria um contêiner de largura total (100%), abrangendo toda a extensão da tela ou dispositivo. Diferente do .container fixo, ele não possui larguras máximas definidas, sendo ideal para layouts responsivos que ocupam a tela inteira, como headera, footers ou seções.
